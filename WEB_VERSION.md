# نسخة الويب من تطبيق تعليم القراءة للأطفال

هذا المجلد يحتوي على نسخة الويب من التطبيق، المستضافة على GitHub Pages.

## الوصول للموقع

رابط الموقع المباشر: `https://ahmadalsaadi.github.io/kidsRead/`

## المحتويات

- **index.html** - الصفحة الرئيسية
- **settings.html** - صفحة الإعدادات
- **css/** - ملفات التنسيق
- **js/** - ملفات JavaScript
- **data/** - قاعدة بيانات الكلمات

## كيفية التطوير محلياً

1. افتح مجلد `docs` في المتصفح مباشرة
2. أو استخدم خادم محلي:
   ```bash
   cd docs
   python3 -m http.server 8000
   ```
3. افتح `http://localhost:8000` في المتصفح

## التحديثات

عند إضافة أي تحديثات، سيتم نشرها تلقائياً على GitHub Pages بعد push إلى الـ repository.
