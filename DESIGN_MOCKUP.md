```

┌──────────────────────────────────────┐

╚════════════════════════════════════════╝
```

## تأثير الحركات على العدد

```
اختيار الحركات → التأثير على العدد:

جميع الحركات     →  250 كلمة
───────────────────────────────

فقط فَتْحَة      →  85 كلمة
───────────────────────────────

فَتْحَة + كَسْرَة →  120 كلمة
───────────────────────────────

فَتْحَة (جميع)   →  210 كلمة
───────────────────────────────

تَنْوِين فقط    →  30 كلمة
```

## تسلسل التفاعلات

```
1️⃣ المستخدم يفتح الإعدادات
   ↓
2️⃣ البرنامج يحمل جميع الكلمات (455 كلمة)
   ↓
3️⃣ عند اختيار طول الكلمة:
   → عداد الطول يتحدث فوراً
   → عداد الحركات يتحدث (تلقائياً)
   ↓
4️⃣ عند تغيير الحركات:
   → عداد الحركات يتحدث فقط
   ↓
5️⃣ المستخدم يرى النتيجة ويقرر:
   ✅ اختيار الإعدادات
   ❌ تعديل الإعدادات
```
